Introduction
============

This Documentation was written for version 2.0.0 of the extension.


What does it do?
----------------


Render images in frontend with area map attached. It supports all features of images like cropVariants. Just remember
that it's not the best idea to wrap images with a link tag. That's why this is disabled in default template.
